---
name: create-from-template
description: 通过替换占位符字符串从模板仓库创建新项目
---

1. 使用 git 命令确定实际的项目名称和仓库位置：
   ```bash
   git remote -v
   ```

2. 阅读项目中的所有文件。理解项目结构。
   使用项目中的可用脚本或任何其他命令行工具查找：

   1. TODO
   2. 需要替换的字符串，例如
      "template"、"YOUR NAME" 以及其他明显的占位符文本。

3. 你应该将这些字符串替换为实际的项目名称。

4. 重新检查项目中的 TODO，确保你已完成所有与项目名称相关的 TODO。
